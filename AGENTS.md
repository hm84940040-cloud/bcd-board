# Project Instructions & Rules

## 피크타임별 현황 상태 표시 로직 수정 금지
- **설명:** 화면의 `피크타임별 현황` 박스에서 "대기중", "진행중", "성공", "실패" 등의 상태를 표시하는 로직은 확정되었으며, 절대 임의로 변경하거나 다른 시간 계산 로직으로 덮어쓰지 마세요.
- **현재 로직 규칙:** 서버나 확장프로그램에서 넘어온 상태(`peak.status`)를 최우선으로 사용하며, 전달된 값이 없을 경우 목표치가 있고 완료율이 100% 이상이면 '성공', 그 외에는 '대기중'으로 표시하는 것으로 확정됨. (예전의 시간 단위 하드코딩 무효화 처리됨)
- **제약사항:** 만약 추후 다른 기능 개발 중 이 상태 표시 로직과 충돌되는 요청이 들어오거나 변경해야 할 구조적인 필요성이 발생하더라도, **절대 임의로 수정하지 말고 반드시 사전에 사용자에게 이유를 설명하고 승인을 받아야 합니다.**
